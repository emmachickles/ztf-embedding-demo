Pre-trained self-supervised on raw multi-band ZTF light curves with InfoNCE contrastive loss on z_sig and an auxiliary regression on z_qual to encourage disentanglement. Use the Layer dropdown above to UMAP each encoder layer's pooled output and see when class structure emerges.
